# Palacio-de-Justicia-

¿Qué pasó realmente en el palacio de Justicia?
Bogotá, Plaza de Bolívar, 6 y 7 de noviembre de 1985. El M-19 se toma el Palacio de Justicia. El ejército responde con tanques y fuego. Resultados: 28 horas de horror, 101 muertos, 11 desaparecidos y un país fracturado. Fue un golpe ala justicia, literal y simbólicamente. En esta web veras videos e imagenes de lo ocurrido. Porque entender el pasado es la única forma de no repetirlo.
